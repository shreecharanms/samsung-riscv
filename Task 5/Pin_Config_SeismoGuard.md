# Pin Configuration:

| Pin	  | Function	   | Component     |
|-------|------------ |---------------|
| D1	   | Shutoff LED	  |Indicator LED  |
| D2	   | Ready LED	  |Indicator LED  |
| D3	   | Buzzer	     |Alert System   |
| D4	   | Relay Input |5V Relay (Bulb)| 
| A1	   | Sensor Data |D7S Sensor     |
| A2    | Sensor Data	|D7S Sensor     |
